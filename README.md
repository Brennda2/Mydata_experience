# 🌐 MyData Experience

O **MyData Experience** é um sistema de conscientização e triagem inteligente de clientes, desenvolvido para criar uma ponte eficiente entre a complexidade técnica dos serviços de tecnologia em nuvem e a experiência do usuário final.

---

## 🚀 Objetivos

* Facilitar o entendimento de conceitos técnicos relacionados à tecnologia em nuvem;
* Auxiliar usuários na identificação de riscos de segurança;
* Apresentar conteúdos educativos de forma clara e interativa;
* Comparar soluções de cloud computing disponíveis no mercado;
* Melhorar a experiência do usuário por meio de uma interface moderna e intuitiva.

## 💻 Principais Funcionalidades

- Início (Home): Apresenta o sistema e traz Flashcards Interativos baseados em casos reais de segurança (como os incidentes da Uber, Equifax, Capital One e o caso do termômetro de Aquário).
- Área Educativa: Um glossário dinâmico explicando conceitos essenciais: Data Center, Cloud Backup, VPN, Omnichannel, Firewall, Malware, VoIP, e Hospedagem. Cada card abre um modal dinâmico focado em soluções.
- Comparativo de Provedores: Uma matriz de comparação técnica e comercial entre a MyData Cloud, AWS e Azure.
- Simulador de Risco: Um formulário de triagem com 4 perguntas estratégicas, calculando o nível de vulnerabilidade (Baixo, Médio ou Alto Risco) emitindo um feedback.
- Quiz Interativo: Um jogo de perguntas e respostas com 10 questões para testar os conhecimentos adquiridos na plataforma, contando com sistema de feedback e pontuação final.

## 🛠️ Tecnologias Utilizadas

- HTML5: Estruturação semântica de layouts e seções.
- CSS3: Estilização moderna, variáveis globais, Grid, Flexbox, animações de transição, efeitos de flip 3D nos cards e responsividade.
- JavaScript (ES6+): Lógica de navegação SPA, manipulação dinâmica do DOM, renderização do gráfico de critérios, cálculo do simulador de risco e motor do quiz estruturado por objetos.

## ⚙️ Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/mydata-experience.git
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd mydata-experience
```

### 3️⃣ Execute a aplicação

Como o projeto utiliza apenas tecnologias front-end nativas, não é necessário instalar dependências adicionais.

Basta abrir o arquivo:

```plaintext
index.html
```

em qualquer navegador moderno:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

Ou utilize a extensão **Live Server** no Visual Studio Code para executar um servidor local.
